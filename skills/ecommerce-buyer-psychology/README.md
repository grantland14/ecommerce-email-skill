# eCommerce Buyer Psychology Skill for Claude

A behavioral science expert for DTC brands, built as a Claude skill. Apply 30 proven psychological principles to your product pages, emails, ads, and pricing to increase conversions ethically.

## What it does

This skill turns Claude into a buyer psychology specialist who connects cognitive science to eCommerce tactics. It covers:

- **30 psychological principles** — Anchoring, loss aversion, social proof, scarcity, the decoy effect, endowment effect, reciprocity, authority, commitment, paradox of choice, framing, peak-end rule, bandwagon effect, contrast principle, Zeigarnik effect, cognitive ease, IKEA effect, halo effect, hyperbolic discounting, and more
- **Practical applications** — Every principle comes with specific eCommerce applications, not just theory
- **Psychology by touchpoint** — How to layer multiple principles on product pages, in emails, and across ads
- **Ethical guidelines** — Clear rules for ethical persuasion vs. manipulation
- **Psychology audits** — Analyze any page or email for psychology gaps and opportunities

## Installation

Copy the skill folder into your Claude Code skills directory:

```bash
git clone https://github.com/grantland14/Claude-Skills-for-eCommerce.git
mkdir -p ~/.claude/skills
cp -rn Claude-Skills-for-eCommerce/skills/ecommerce-buyer-psychology ~/.claude/skills/
```

Start a new Claude Code conversation — Claude will automatically use this skill when relevant.
