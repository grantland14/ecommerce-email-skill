# eCommerce PDP Optimization Skill for Claude

A product detail page conversion expert for DTC brands, built as a Claude skill. Analyze and improve product pages element by element to increase add-to-cart rates, reduce bounce, and turn browsers into buyers.

## What it does

This skill turns Claude into a PDP conversion specialist who understands how eCommerce customers evaluate products and make buying decisions. It covers:

- **The 5 buyer questions** — Framework for evaluating whether your PDP answers what customers need to know
- **Above-the-fold audit** — Checklist for desktop and mobile above-the-fold optimization
- **Product image strategy** — Optimal 7-10 image sequence and quality standards
- **Below-the-fold content** — Section order, content blocks, and FAQ strategy
- **Cross-sell and upsell** — AOV-increasing strategies on the product page
- **Mobile optimization** — Sticky CTAs, swipeable galleries, touch-friendly design
- **Category-specific guidance** — Beauty, fashion, food, home, supplements
- **Full PDP audit** — Comprehensive checklist with prioritized recommendations

## Installation

Copy the skill folder into your Claude Code skills directory:

```bash
git clone https://github.com/grantland14/Claude-Skills-for-eCommerce.git
mkdir -p ~/.claude/skills
cp -rn Claude-Skills-for-eCommerce/skills/ecommerce-pdp ~/.claude/skills/
```

Start a new Claude Code conversation — Claude will automatically use this skill when relevant.

## Example prompts

- *"Audit my product page — here's the URL. What's killing my conversion rate?"*
- *"Rewrite my product description for this moisturizer — currently it's just a list of ingredients"*
- *"What should my product image gallery look like for a $120 face serum?"*

- *"What cross-sell strategy should I use on my PDPs to increase AOV?"*

## Who this is for

DTC founders who know their product pages should convert better but don't know exactly what to fix. Get specific, prioritized recommendations that drive real revenue improvement.
