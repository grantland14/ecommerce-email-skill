# eCommerce Popup Optimization Skill for Claude

An expert popup strategist for DTC brands, built as a Claude skill. Design high-converting welcome popups, exit-intent offers, promotional popups, and announcement bars that grow your email list and drive revenue.

## What it does

This skill turns Claude into a popup conversion specialist who knows what works in eCommerce. It covers:

- **Welcome popups** — Email and SMS capture for new visitors
- **Exit-intent popups** — Last-chance offers before visitors leave
- **Promotional popups** — Sales, bundles, and limited-time offers
- **Announcement bars** — Sitewide messaging for shipping, sales, and launches
- **Cart upsell popups** — Increase AOV at the point of purchase
- **Back-in-stock popups** — Capture demand for sold-out products
- **Multi-popup strategy** — Coordinating multiple popups without annoying visitors
- **Copy formulas** — Headlines, body copy, and CTAs proven to convert
- **Platform guidance** — Klaviyo, Privy, Justuno, and OptiMonk setup tips

## Installation

Copy the skill folder into your Claude Code skills directory:

```bash
git clone https://github.com/grantland14/Claude-Skills-for-eCommerce.git
mkdir -p ~/.claude/skills
cp -rn Claude-Skills-for-eCommerce/skills/ecommerce-popup ~/.claude/skills/
```

Start a new Claude Code conversation — Claude will automatically use this skill when relevant.
